Dataset Description

The MobileCN dataset provides granular telecommunication data aggregated into 500m x 500m spatial grids. The data is anonymized to ensure user privacy while maintaining statistical utility for urban planning and network optimization.

Data Format: CSV / NPZ

Field Name Data Type Description Example
grid_id String Unique identifier for the spatial grid (500m resolution). G_1024_503
timestamp DateTime Time of record (15-minute intervals). 2025-12-01 08:15:00
user_count Integer Number of distinct active users in the grid. 452
traffic_ul Float Total upload traffic volume (GB). 12.5
traffic_dl Float Total download traffic volume (GB). 89.3
app_category String Dominant application type (e.g., Video, Social, Game). Video_Streaming
