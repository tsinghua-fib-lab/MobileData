Dataset Description

The MobileCN dataset provides granular telecommunication data aggregated into 500m x 500m spatial grids. The data is anonymized to ensure user privacy while maintaining statistical utility for urban planning and network optimization.

Data Format: CSV / NPZ

Field Name Data Type Description Example
grid_id Integer Unique identifier for the spatial grid (500m resolution). 16
lat, lon Float Location (WGS 84 coordinate system). [39.453, 116.35]
data_500m Float MetaData (Mobile traffic or Actie user, 1-hour intervals, 168-length). [43.9,1.39, 9.16,..., 54.2]
poi_500m Integer Point of Interest (21-length) [0 0 2 0 0 4 0 5 0 0 12 0 0 4 0 0 0 11 0 0 2]
water_area_500m Float Water area (meters). 249918.96
roadlen_500m Float Road length (meters). 2258.59
building_500m Integer Building numbers. 12
pop_500m Float Resident population size. 216.082
